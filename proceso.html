Idea clave

No trabajamos con promesas abiertas del tipo “ya veremos”. Definimos bien el alcance para que sepas qué compras, qué puedes esperar y cuándo hace falta subir de nivel.

01

¿Los precios son cerrados?

Sí, cuando el alcance está bien definido. Web Lite, Web Express y Web Pro tienen una base clara. Web Personalizada parte desde un precio orientativo, pero necesita presupuesto según funcionalidades, integraciones y complejidad real.

  • Web Lite: 200€ + IVA
  • Web Express: 350€ + IVA
  • Web Pro: 600€ + IVA
  • Web Personalizada: desde 600€ + IVA
02

¿Cuánto tarda cada tipo de web?

Depende del material disponible y del alcance. Web Lite puede moverse rápido si ya tenemos textos, fotos y estructura clara. En Express, Pro y Personalizada el plazo se confirma al cerrar contenidos, revisiones y complejidad.

  • Lite: más ágil cuando el material está listo
  • Express y Pro: plazo acordado al inicio
  • Personalizada: según módulos, lógica y desarrollo
03

¿Qué significa SEO básico, medio o avanzado inicial?

Significa preparar la web para salir bien publicada, con buena estructura, indexación y buenas prácticas. No significa prometer posiciones en Google ni vender posicionamiento continuo como si fuera automático.

  • Lite: SEO básico inicial
  • Express: estructura y SEO medio inicial
  • Pro: SEO avanzado inicial y optimización técnica
04

¿Dominio y hosting están incluidos?

Depende del plan. En Lite y Express el dominio no está incluido y el hosting va aparte. En Pro y Personalizada el primer año está incluido porque ya forman parte de una estructura más completa.

  • Lite: hosting 35€/año, dominio aparte
  • Express: hosting 50€/año, dominio aparte
  • Pro: primer año incluido; renovación 60€ hosting + 15€ dominio
  • Personalizada: primer año incluido; renovación desde 80€ hosting + 15€ dominio
05

¿Puedo actualizar la web después?

Sí, pero no todos los planes lo resuelven igual. En Web Pro hay autogestión básica de ciertos contenidos. En el resto, los cambios posteriores pueden entrar como mantenimiento o bono de horas según lo que necesites.

  • Mantenimiento mensual para seguimiento continuo
  • Bonos para acciones concretas
  • Web Pro con panel para bloques definidos
06

¿Cuándo encaja Web Pro y cuándo pasa a Web Personalizada?

Web Pro sigue siendo una web corporativa avanzada: capta contactos, gestiona ciertos contenidos y permite medir lo básico. Cuando ya necesitas reservas, áreas privadas, automatizaciones, CRM, integraciones complejas o lógica específica del negocio, el proyecto pasa a Web Personalizada.

  • Web Pro: panel para noticias, eventos, promociones o casos de éxito
  • Web Pro: formulario con almacenamiento de contactos
  • Web Pro: gestión básica de contenidos, no procesos internos
Salto de categoría

Si el proyecto necesita reservas, usuarios, dashboards, automatizaciones, áreas privadas, integraciones o herramientas internas, deja de ser una web corporativa cerrada y pasa a presupuesto personalizado.

Más dudas comunes

Preguntas rápidas que suelen salir en la primera conversación

¿Necesito tener ya todos los textos preparados?

No siempre. En Lite y Express podemos ayudarte a ordenar la información base. En Pro y Personalizada conviene definir mejor materiales, accesos y objetivos para no perder tiempo durante el desarrollo.

¿Puedo pedir varios idiomas?

Sí, pero no todos los proyectos lo necesitan ni todos lo incluyen de base. Si la web va a tener varios idiomas reales, se valora como parte del alcance en Express, Pro o Personalizada.

¿La web se verá bien en móvil?

Sí. El responsive no es un extra: forma parte del estándar mínimo de trabajo en todos los planes.

¿Puedo empezar con algo pequeño y crecer después?

Sí, y de hecho suele ser la forma más sensata. Puedes arrancar con Web Lite o Web Express y, cuando el negocio lo pida de verdad, pasar a Pro o a una solución personalizada.

¿Tienes otra duda concreta?

La primera conversación sirve precisamente para aterrizar el proyecto y decirte con claridad qué opción encaja mejor con tu negocio.

Qué es este proyecto

Der Die Das se planteó como un microproducto dentro de una familia educativa más amplia. La prioridad fue velocidad de uso, una interfaz muy directa y una experiencia casi de repetición instantánea, ideal para sesiones cortas y frecuentes.

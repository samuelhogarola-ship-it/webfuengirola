# Web Project Questionnaire

Cuestionario corto para usar siempre antes de crear una preview o empezar una web.

## Preguntas base

1. `Que modelo necesitas?`
   `Lite / Express / Personalizada / Web Pro / App`

2. `Que tipo de hero quieres?`
   `Hero fijo Lite / Modelo Clueso / Full-screen reveal / Split premium / Cards commercial / Editorial minimal / Video light`

3. `Que estilo visual manda?`
   `Premium minimalista / Editorial / Nike-Porsche agresivo limpio / Blog de viajes / Tienda generica / Otro`

4. `La web sera monopage o multipage?`

5. `Necesita blog?`
   `Si / no`

6. `Necesita eventos?`
   `Si / no`

7. `Necesita tienda o venta online?`
   `Si / no`

8. `Quieres banner de cookies personalizado o estandar?`
   `Personalizado / estandar / no aplica`

9. `Cual es el CTA principal?`
   Ejemplos: `agendar primera cita`, `pedir presupuesto`, `WhatsApp`, `comprar`

10. `Que paginas o secciones son obligatorias?`
    Ejemplos: `inicio`, `servicios`, `entrenos`, `contacto`, `legal`

11. `Que assets reales tenemos ya?`
    `logo / fotos / textos / referencias / paleta`

12. `Que no debe pasar visualmente?`
    Aqui fijamos limites antes de disenar.

## Nota de patron

- `Modelo Clueso`
  Hero-portada con imagen persistente durante la bajada inicial y contenido entrando por encima sin rail lateral ni sticky visible.
  Alias interno recomendado: `lite-clueso`

## Cierre

Con estas respuestas ya se puede:

- elegir estructura
- definir direccion visual
- abrir ruta de preview
- arrancar propuesta con criterio

# Madame Bleue | Monaco

Local landing page concept preview for Madame Bleue Watches.

## Status

This project is a local preview only. It is not official, not production, and not a deployed website.

The preview now integrates the Madame Bleue logo and product photography supplied as authorized demo assets. It remains a concept preview and must not be treated as a production website without final client review.

## Stack

- Vite
- React
- TypeScript
- Custom CSS

No backend, CMS, checkout, cart, login, payments, Supabase, or external integrations are included.

## Assets

Logo path:

- `/images/brand/logo-madame-1024x1024.png`

Product image folders:

- `/images/products/patek-philippe-5146j/`
- `/images/products/rolex-gmt-master-ii/`
- `/images/products/panerai-tuttonero-pam01438/`
- `/images/products/cartier-santos-dumont-w2sa0011/`

Product image naming convention:

- Use `01.jpg`, `02.jpg`, `03.jpg`, etc.
- `01.jpg` is the primary image for each product.
- Real product photography in these folders has been added as authorized demo material for Madame Bleue.
- Future photography should only be used when authorized by Madame Bleue.

## Run Locally

```bash
npm install
npm run dev
```

The local dev server will print the preview URL in the terminal.

## Build

```bash
npm run build
```

## Notes

- The page includes `noindex,nofollow` metadata.
- It contains no public pricing and no ecommerce flow.
- CTAs scroll to local sections only.
- Instagram is presented as a visual external link.
- Contact details remain placeholders pending approved client information.
- Product and brand visuals are local demo assets, not production-ready launch material.
